# Instruction on how to version 

Version number is put in the bottom left corner of the app

The app version follow this format each time a file is edited within the project.

## The format

YY.MM.DD 

For example, a file is edited on 31st March 2026, then version will be 26.3.31. Display "Version 26.3.31"
